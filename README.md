# API Debugging Starter Pack (Beginner)

This repo is a beginner-friendly practice lab for technical product support.
It includes:
- A simple API debugging runbook
- Reproducible examples using curl and Python
- Support-style ticket notes (what to collect, how to escalate)

## What I’m practicing
- Reading API requests/responses (headers, status codes, body)
- Common failures: 404, 401/403, 429 (rate limits)
- Writing clear support notes and escalation-ready repro steps

## Repo map
- `runbooks/` → step-by-step debugging guides
- `examples/` → working request examples (curl + Python)
- `ticket-notes/` → realistic support tickets and templates

## APIs used (public)
- JSONPlaceholder (practice API)
- GitHub API (public endpoints + rate limit behavior)
